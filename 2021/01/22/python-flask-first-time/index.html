原因#

套件管理 pip#

pip 基本上可以當做 php 的 composer，
相關的紀錄會在 requirements.txt 。

指令#

如果有更動過套件，記得在專案下 pip freeze > requirements.txt，把目前的套件記錄匯出來，避免套件不一致。

這邊他也會把相依的套件記錄也一併輸出。
